# Cyber Forensics
Tamper detection and integrity over chain of custody
# Objectives

This project aims to develop a secure web-based Digital Forensic Evidence Management System that ensures integrity, traceability, and controlled distribution of digital evidence.

The system achieves this by:

-> Generating SHA-256 and fuzzy hashes during file upload

-> Re-validating hashes during every download

-> Maintaining secure, tamper-evident audit logs

-> Implementing role-based access control

-> Preserving digital chain-of-custody records

-> Detecting partial or complete evidence modifications
